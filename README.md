Запустить `first_run.sh` для инициализации базы!

После этого в локальном файле `tsserver.sqlitedb` будет хранится база данных сервера, и она будет монтироваться в контейнер при всех последующих перезапусков.

Admin Token будет в выводе скрипта `first_run.sh`

Дальнейшие запуски просто через `docker compose up -d`
